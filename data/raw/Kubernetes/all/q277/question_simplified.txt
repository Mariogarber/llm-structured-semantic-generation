Create a K8s StatefulSet for a Cassandra cluster with resource limits/requests: CPU limit 500m, Memory limit 1Gi, CPU request 500m, Memory request 1Gi. Here's a half-done YAML:
```
apiVersion: apps/v1
kind: StatefulSet
metadata:
  name: my-cassandra
  labels:
    app: my-cassandra
spec:
  serviceName: cassandra
  replicas: 3
  selector:
    matchLabels:
      app: my-cassandra
  template:
    metadata:
      labels:
        app: my-cassandra
    spec:
      containers:
      - name: cassandra
        image: gcr.io/google-samples/cassandra:v13
        env:
          - name: MAX_HEAP_SIZE
            value: 512M
          - name: HEAP_NEWSIZE
            value: 100M
```
