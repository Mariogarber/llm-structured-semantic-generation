这是一个关于Kubernetes的问题。
我有两个ConfigMap的YAML文件。你能使用它们在同一个busybox Pod中创建多个文件吗？将它们挂载到"/etc"下。
```
apiVersion: v1
kind: ConfigMap
metadata:
  name: config1
data:
  key1: "value1"

apiVersion: v1
kind: ConfigMap
metadata:
  name: config2
data:
  key2: "value2"
```
对于容器命令的一些额外提示：
```
["sh", "-c", "sleep 1200"]
```
