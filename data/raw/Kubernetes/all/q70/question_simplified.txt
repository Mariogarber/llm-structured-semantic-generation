Make changes to k8s Deployment manifest:
- Change Deployment name to my-deploy.
- Increase replicas by 2.
- Use nginx image version 1.16.1.
- Set port to 8080.
```
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
spec:
  selector:
    matchLabels:
      app: nginx
  replicas: 1
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx:1.14.2
        ports:
        - containerPort: 80
```
