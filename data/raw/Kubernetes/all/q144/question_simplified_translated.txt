为 Pod 中的所有容器指定安全设置。Pod 中的容器应具有以下设置：
- 所有进程以用户 ID 1000 运行。
- Pod 中任何容器内的所有进程的主组 ID 为 3000。
- 容器的所有进程都属于附加组 ID 2000。
- 容器的所有进程不能获得比其父进程更多的特权。

```
apiVersion: v1
kind: Pod
metadata:
  name: security-context-demo
spec:
  volumes:
  - name: sec-ctx-vol
    emptyDir: {}
  containers:
  - name: sec-ctx-demo
    image: busybox:1.28
    command: [ "sh", "-c", "sleep 1h" ]
    volumeMounts:
    - name: sec-ctx-vol
      mountPath: /data/demo
```
