Modify this Pod to have 2 Init Containers. The first, named init-myservice, waits for Service: myservice to start. The second, named init-mydb, waits for Service: mydb to start. Once both Init containers have started, the Pod will start the application container named myapp-container. All containers use image: busybox:1.28.
```
apiVersion: v1
kind: Pod
metadata:
  name: myapp-pod
  labels:
    app.kubernetes.io/name: MyApp
spec:
  containers:
  - name: myapp-container
    image: busybox:1.28
    command: ['sh', '-c', 'echo The app is running! && sleep 3600']
```
