更新 Kubernetes StatefulSet "my-cassandra"，包括一个名为 "fast" 的 StorageClass，使用 "k8s.io/minikube-hostpath" 作为 provisioner。对于 PersistentVolumeClaim，请求 100MB 存储，并使用 ReadWriteOnce 访问模式。
```
apiVersion: apps/v1
kind: StatefulSet
metadata:
  name: my-cassandra
  labels:
    app: my-cassandra
spec:
  serviceName: cassandra
  replicas: 3
  selector:
    matchLabels:
      app: my-cassandra
  template:
    metadata:
      labels:
        app: my-cassandra
    spec:
      containers:
      - name: cassandra
        image: gcr.io/google-samples/cassandra:v13
        env:
          - name: MAX_HEAP_SIZE
            value: 512M
          - name: HEAP_NEWSIZE
            value: 100M
```
