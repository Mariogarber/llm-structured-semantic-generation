k8s 可以使用文件中的 env var 而不是硬编码吗？ 给定：
```
apiVersion: v1
kind: Pod
metadata: 
  labels: 
    context: docker-k8s-lab
    name: mysql-pod
  name: mysql-pod
spec: 
  containers: 
    - env: 
        - name: MYSQL_USER
          value: mysql
        - name: MYSQL_PASSWORD
          value: mysql
        - name: MYSQL_DATABASE
          value: sample
        - name: MYSQL_ROOT_PASSWORD
          value: supersecret
      image: "mysql:latest"
      name: mysql
      ports: 
        - containerPort: 3306
```
假设一个名为 mysql-secret 的 Secret 含有所有值。 提供完整的 YAML。
