你需要设置一个 Kubernetes Job，它将生成 3 个并行的 Pod（example-job）。这些 Pod 应该能够使用 DNS 相互通信。为了实现这一点，你还需要创建一个无头服务（headless-svc）。作业的 Pod 应该通过它们的 DNS 名称相互 ping（提供的脚本）。使用 bash:latest 镜像作为作业的容器。
```
for i in 0 1 2
do
  gotStatus="-1"
  wantStatus="0"             
  while [ $gotStatus -ne $wantStatus ]
  do                                       
    ping -c 1 example-job-${i}.headless-svc > /dev/null 2>&1
    gotStatus=$?                
    if [ $gotStatus -ne $wantStatus ]; then
      echo "Failed to ping pod example-job-${i}.headless-svc, retrying in 1 second..."
      sleep 1
    fi
  done                                                         
  echo "Successfully pinged pod: example-job-${i}.headless-svc"
done
```
