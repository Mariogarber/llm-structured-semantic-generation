创建一个包含特定数据的 ConfigMap，并在 DaemonSet 中使用它来每隔10秒连续打印数据。使用以下的 DaemonSet 结构：

```
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: ds-configmap-volume
spec:
  selector:
    matchLabels:
      name: ds-configmap-volume
  template:
    metadata:
      labels:
        name: ds-configmap-volume
    spec:
      containers:
      - name: ds-configmap-container
        image: busybox
        command: [ "sh", "-c", "while true; do cat /config/config-data.txt; sleep 10; done" ]
        volumeMounts:
        - mountPath: /config
          name: config-volume
      volumes:
      - name: config-volume
        configMap:
          name: ds-configmap
```
