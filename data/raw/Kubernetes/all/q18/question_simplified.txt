Add a nodeSelector to the given DaemonSet YAML to ensure Pods run only on nodes labeled "disktype=ssd".
```
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: nodeselector-daemonset
spec:
  selector:
    matchLabels:
      app: nodeselector-daemonset
  template:
    metadata:
      labels:
        app: nodeselector-daemonset
    spec:
      containers:
      - name: busybox
        image: busybox
        command: ["sleep", "3600"]
```
