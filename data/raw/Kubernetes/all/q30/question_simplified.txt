Add an env var named GREETING_MESSAGE with value "Hello from the environment variable!" to the container spec in the given YAML.
```
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: ds-env-vars
spec:
  selector:
    matchLabels:
      name: ds-env-vars
  template:
    metadata:
      labels:
        name: ds-env-vars
    spec:
      containers:
      - name: ds-env-container
        image: busybox
        command: [ "sh", "-c", "while true; do echo $GREETING_MESSAGE; sleep 10; done" ]
```
