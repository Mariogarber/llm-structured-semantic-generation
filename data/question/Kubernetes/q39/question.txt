Given the following YAML that created a DaemonSet, please help me:
1. Add two containers to the specification. The first container should be named "container-main", use the busybox image, and execute the 
command "while true; do echo 'Main container'; sleep 10; done". The second container should be named "container-sidecar", also use 
the "busybox" image, and execute the command "while true; do ps aux; sleep 15; done".
2. Ensure that the containers within this DaemonSet share the same process namespace by setting the appropriate field to true.
```
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: ds-share-process-ns
spec:
  selector:
    matchLabels:
      name: ds-share-process-ns
  template:
    metadata:
      labels:
        name: ds-share-process-ns
    spec:
      containers:
      - name: container-main
        image: busybox
        command: [ "sh", "-c", "while true; do echo 'Main container'; sleep 10; done" ]
      - name: container-sidecar
        image: busybox
        command: [ "sh", "-c", "while true; do ps aux; sleep 15; done" ]
```
