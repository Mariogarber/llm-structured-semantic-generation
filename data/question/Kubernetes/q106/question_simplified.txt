Modify the Kubernetes Job config to ignore pods that fail with status code 0. No job failure or retries for such pods. Other pods should continue execution.
```
apiVersion: batch/v1
kind: Job
metadata:
  name: ignored-failjob
spec:
  completions: 4
  parallelism: 1
  template:
    spec:
      restartPolicy: Never
      containers:
      - name: main
        image: alpine
        command:
        - "sh"
        - "-c"
        - "sleep 5"
```
