Expose container ports 7000 as intra-node and 9042 as cql in the existing StatefulSet YAML for my-cassandra.
```
apiVersion: apps/v1
kind: StatefulSet
metadata:
  name: my-cassandra
  labels:
    app: my-cassandra
spec:
  serviceName: cassandra
  replicas: 3
  selector:
    matchLabels:
      app: my-cassandra
  template:
    metadata:
      labels:
        app: my-cassandra
    spec:
      containers:
      - name: cassandra
        image: gcr.io/google-samples/cassandra:v13
        env:
          - name: MAX_HEAP_SIZE
            value: 512M
          - name: HEAP_NEWSIZE
            value: 100M
```
