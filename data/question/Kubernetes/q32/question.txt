Given a Kubernetes structure, I want to create a ConfigMap that holds specific data. I aim to utilize it in a DaemonSet that will 
continuously print the data from this ConfigMap every 10 seconds. For reference, consider the DaemonSet structure below:
```
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: ds-configmap-volume
spec:
  selector:
    matchLabels:
      name: ds-configmap-volume
  template:
    metadata:
      labels:
        name: ds-configmap-volume
    spec:
      containers:
      - name: ds-configmap-container
        image: busybox
        command: [ "sh", "-c", "while true; do cat /config/config-data.txt; sleep 10; done" ]
        volumeMounts:
        - mountPath: /config
          name: config-volume
      volumes:
      - name: config-volume
        configMap:
          name: ds-configmap
```
