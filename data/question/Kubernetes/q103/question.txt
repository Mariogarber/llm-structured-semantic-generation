You need to set up a Kubernetes Job that will spawn 3 parallel pods (example-job). These pods should be able to communicate with each other using DNS. To facilitate this, you also need to create a headless service (headless-svc). The job's pods should ping each other by their DNS names (script provided). Use the bash:latest image for the job's container.
```
for i in 0 1 2
do
  gotStatus="-1"
  wantStatus="0"             
  while [ $gotStatus -ne $wantStatus ]
  do                                       
    ping -c 1 example-job-${i}.headless-svc > /dev/null 2>&1
    gotStatus=$?                
    if [ $gotStatus -ne $wantStatus ]; then
      echo "Failed to ping pod example-job-${i}.headless-svc, retrying in 1 second..."
      sleep 1
    fi
  done                                                         
  echo "Successfully pinged pod: example-job-${i}.headless-svc"
done
```
