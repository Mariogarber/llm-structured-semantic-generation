How to set the storage request to 1,000,000,000 bytes in the given code?
```
apiVersion: apps/v1
kind: StatefulSet
metadata:
  name: pg-db-storage
spec:
  serviceName: "pg-service"
  replicas: 1
  selector:
    matchLabels:
      app: pg-db
  template:
    metadata:
      labels:
        app: pg-db
    spec:
      containers:
      - name: postgresql
        image: postgres:latest
        volumeMounts:
        - name: pgdata
          mountPath: /var/lib/postgresql/data
  volumeClaimTemplates:
  - metadata:
      name: pgdata
    spec:
      accessModes: ["ReadWriteOnce"]
      resources:
        requests:
          storage: 1Gi
```
