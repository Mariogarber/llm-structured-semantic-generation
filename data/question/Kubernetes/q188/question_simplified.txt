Use the configuration item in the volume by specifying the name as "config-volume" under Volumes. The key-value pairs are in the config file named "special-config". Store the data of key-values in the MountPath path "/etc/config".
```
apiVersion: v1
kind: Pod
metadata:
   name: config-pod-4
spec:
   containers:
     - name: test-container
       image: mirror.gcr.io/library/busybox:latest
       command: [ "/bin/sh", "-c", "while true; do sleep 3600; done" ]
```
