In Kubernetes, I want to create a ServiceAccount named sample-sa in the kube-system namespace. I also want to create a ClusterRoleBinding named sample-clusterrolebinding in the same namespace. This ClusterRoleBinding should bind the ClusterRole named sample-clusterrole to the ServiceAccount sample-sa.
The manifests for ServiceAccount and ClusterRole are provided below. You can add them together in the final YAML.
```
apiVersion: v1
kind: ServiceAccount
metadata:
  name: sample-sa
  namespace: kube-system

apiVersion: rbac.authorization.k8s.io/v1
kind: ClusterRole
metadata:
  name: buggy-clusterrole
rules:
  - apiGroups: [""]
    resources: ["pods", "services"]
    verbs: ["get", "list", "watch"]
  - apiGroups: ["apps"]
    resources: ["deployments"]
    verbs: ["get", "list"]
```
