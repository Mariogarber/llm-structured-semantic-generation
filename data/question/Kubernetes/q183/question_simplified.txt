Specify security settings for all Containers in the pod. Set the group ID as 1234 that owns the volume and can write to it. Use the following YAML:

```
apiVersion: v1
kind: Pod
metadata:
  name: security-context-demo-2
spec:
  volumes:
  - name: sec-ctx-vol
    emptyDir: {}
  containers:
  - name: sec-ctx-demo
    image: busybox:1.28
    command: [ "sh", "-c", "sleep 1h" ]
    volumeMounts:
    - name: sec-ctx-vol
      mountPath: /data/demo
```
