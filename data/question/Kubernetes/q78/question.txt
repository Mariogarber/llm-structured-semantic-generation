Given the following redis leader deployed:
```
apiVersion: apps/v1
kind: Deployment
metadata:
  name: redis-leader
  labels:
    app: redis
    role: leader
    tier: backend
spec:
  replicas: 1
  selector:
    matchLabels:
      app: redis
  template:
    metadata:
      labels:
        app: redis
        role: leader
        tier: backend
    spec:
      containers:
      - name: leader
        image: "docker.io/redis:6.0.5"
        resources:
          requests:
            cpu: 100m
            memory: 100Mi
        ports:
        - containerPort: 6379
```
Please write a k8s YAML file to make it highly available and meet traffic demands by deploying 2 redis followers. 
Please use image "gcr.io/google_samples/gb-redis-follower:v2". 
The deployment and pods should be labeled with "app: redis", "role: follower" and "tier: backend".
