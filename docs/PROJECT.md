# Historical Project Note

This document is kept as a historical note for the early thesis framing.

The original project formulation used `docker-compose.yaml` as the case study for
structured YAML generation. The current operative repository scope has moved to
Kubernetes manifest generation, with `Kubernetes v1` as the fixed dataset for the
main experimental path.

Current source-of-truth documents:

- `README.md`
- `AGENTS.md`
- `docs/KUBERNETES_PREPROCESSING.md`
- `docs/KUBERNETES_MODEL_V1.md`
- `docs/STRUCTURAL_TARGETS_V1.md`
- `docs/BASELINE_V1.md`

Historical Docker Compose motivation may still be useful as thesis background,
but it must not override the current Kubernetes-centered implementation.
